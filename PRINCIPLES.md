# 번역 규칙
첫번째로 해당 용어가 출현하였을 때는 '한글(영어)'로 번역하고, 이후부터는 한글로만 번역합니다. (예. including transposing, indexing, ... => 전치(transposing), 인덱싱(indexing), ...)
